# CS2 Translation Assistant

Переводчик чата для Counter-Strike 2 в реальном времени — для русскоязычных
игроков, которые устали не понимать иностранных тиммейтов и соперников (и
не хотят переключаться в браузер посреди матча).

## Что делает

- **Переводит входящий чат автоматически.** Кто-то написал не по-русски —
  рядом с чатом всплывает перевод, с коротким звуковым сигналом, чтобы не
  пропустить, пока вы целитесь.
- **Помогает отвечать.** Хоткей открывает маленькое окно — печатаете по-русски,
  Enter — перевод сам уходит в чат игры (отдельные хоткеи для команды и
  общего чата, чтобы случайно не перепутать канал).
- **Не мешает играть.** Оверлей прозрачный и не перехватывает клики/фокус,
  сам скрывается, когда вы в меню, а не в матче.

## Почему это безопасно для аккаунта

Никакого чтения памяти процесса игры, никаких инъекций, никакой модификации
файлов игры (кроме одного официального конфига CS2 Game State Integration —
задокументированная функция Valve для сторонних оверлеев/трансляций, ею же
пользуются HUD-виджеты и инструменты казуальных трансляций).

- Входящий чат читается из `console.log`, который сама игра пишет при
  запуске с `-condebug` — обычный текстовый файл, не память процесса.
- Ответы отправляются через буфер обмена и эмуляцию нажатий клавиш — то же
  самое, что делаете вы сами, когда печатаете в чат руками.

На площадках с обязательным сторонним античитом (FACEIT и т.п.) автоотправка
в чат отключается сама — перевод просто копируется в буфер, вставляете
вручную, как обычно.

## Установка

1. Скачайте установщик из [Releases](https://github.com/YaroCore/cs2-translation-assistant-releases/releases/latest).
2. Запустите **от имени администратора**, введите лицензионный ключ (пока подходит любой — идёт обкатка).
3. В открывшемся окне нажмите "Запустить ассистента".
4. В CS2: Параметры запуска → добавить `-condebug`, режим экрана → "Оконный (без рамки)".

Подробная инструкция, хоткеи и ответы на частые вопросы (в том числе "Windows
не доверяет программе" и "хоткей не срабатывает") — в
[INSTRUCTIONS.md](INSTRUCTIONS.md).

Программа обновляется сама — при выходе новой версии в трее появится
уведомление с кнопкой "Обновить".
